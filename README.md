## Description

`@ossph/opensource` is a utility for generating Open Source Projects. It provides a toolset for scaffolding project structures and setting up common configurations.

## Installation

You can install the package using npm:

```sh
npm install @ossph/opensource
```

## Usage

Once installed, you can use the utility to generate an Open Source Project by running the following command:

```sh
npx @ossph/opensource
```

This will prompt you for project details and then generate the necessary files and configurations based on your input.

## Scripts

The following npm scripts are available:

- `test`: Run tests using Jest.
- `publish`: Publish the package to npm with public access.
- `build`: Build the project using Rollup.

You can run these scripts using npm. For example, to run tests:

npm test
