# Open Source Project Generator

An easy to use boiler plate for open source projects with the following features:

- [x] README.md
- [x] LICENSE
- [x] CONTRIBUTING.md
- [x] CHANGELOG.md
- [x] CODE_OF_CONDUCT.md

## Usage 

`npx @ossph/opensource <project-name>`
